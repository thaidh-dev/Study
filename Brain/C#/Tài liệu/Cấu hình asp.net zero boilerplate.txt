appsetting.json: khai báo tên database, username, pass

launchSetting.json: khai báo url của ứng dụng
		    khai báo môi trường hoạt động

wwwroot: đặt file vào trong đấy

project.assets.json: hình như là file cấu hình

request pipeline: là cả cái đường mà requets đi lên server rồi đi về

.core: entity nằm trong đấy

.EntityFrameworkCore.EntityFrameworkCore: chứa dbContext

.Application: chứa controller (bắt buộc phải extends QuanLyThietBiAppServiceBase và phải có method thì mới có class trong service-proxies.ts)              

sidebar-nav.component.ts: file lấy đường dẫn màn hình

app-routing.module.ts: lấy đường dẫn component

service-proxy.module.ts: khai báo service proxy
